Extensiones que hay que instalar:
1-MOngoose: para la conexion y trabajo con la base de Datos Mongodb
2-Express: para trabajar mejor los metodos del backend y no hacerlo con Node puro
3-Cors: permite decirle al backend(api nuestra) los que se pueden conectar a ella


Como extension para el desarrollo
1-nodemon para que reinicie el servidor cada vez que se guarda algun cambio
2-eslint: para darle formato al codigo se instala con npm install eslint -D y se configura .\node_modules\.bin\eslint --init. 
Luego en el Visual Studio Code se debe istalar la extension EsLint


Ext en Visual Studio Code.
1-Rest Client sirve para simular las conexiones 
2-Error Lens muestra los errores por liniea
